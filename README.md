## 🚀 What bashguy does

bashguy is a shell widget that turns plain English into bash commands with Claude.

Type what you want to do, and bashguy helps you shape it into a command you can use in your shell. It is meant for people who want faster command-line work without memorizing every flag and option.

Use it to:
- turn a request into a bash command
- save time when you forget syntax
- work with the command line in a simpler way
- get command ideas before you run them

## 🧑‍💻 How to use it

Use bashguy like a helper for command writing.

Try prompts such as:
- show me files larger than 100 mb
- find all .log files in this folder
- list running processes using port 8080
- create a zip file for this directory
- search for the word error in all text files

bashguy will use your request and turn it into a bash command. You can then review the command before you use it in your shell.

## 🔧 Basic workflow

1. Open bashguy.
2. Type what you want to do in plain language.
3. Let it generate a bash command.
4. Check the command.
5. Copy it into your shell.
6. Run it if it looks right.

This flow helps you work faster while keeping control of the command you run.

## 📌 Tips for better results

Use short, clear requests.

Good:
- find large files in downloads
- show git status for this folder
- compress the current directory

Less clear:
- do the thing with the files in that place

If you include the folder, file type, or result you want, bashguy can build a better command.

## 🛠️ Troubleshooting
